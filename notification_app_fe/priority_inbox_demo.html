⚙️ Algorithm: Min-Heap of Size N

Each notification gets a composite score: score = typeWeight × 10¹³ + timestamp(ms). Type weights: placement=3, result=2, event=1. We maintain a min-heap of size N — for each new notification, if its score beats the heap's minimum it replaces it. This gives O(M log N) time over M notifications, scaling efficiently as the stream grows.
